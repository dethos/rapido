How Rapido handles the data?

Given that the objective of this aplication is to create public and shareable TODOs, all data created through it's usage (tasks, lists and progress entries) is publicly accessible to anyone with access to the list's URL or identification.

The only exception to this rule is the user's login information that is kept secret (and is not required).
